Чтобы начать работу бота нужно:
1. Добавить свои данные Kaspi, данные нужно будет ввести в одном сообщении, через запятую(логин, пароль), нажмите кнопку "Добавить данные"
2. Добавить таблицу с данными товаров для обработки(вормат xlsx), есть 4 основных колонки: Артикул, описание, цена, минимальная цена. Если с первыми тремя все понятно, то минимальная цена может быть просто скопированна с основной цены (таблица должна выглядеть примерно так). Нажмите "Мои товары" -> "Добавить товар"
